# Docker for Ruby development.
It is based on this [article](https://evilmartians.com/chronicles/reusable-development-containers-with-docker-compose-and-dip)

## Usage

Clone this repository as `~/.dip/`. And run `make`

```shell
cd ~
clone repo-url .dip
cd .dip && make
cd your_project
dip ls
dip ruby
ruby -v
```
